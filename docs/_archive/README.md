# _archive

> Historical / scratch — correspondence, drafts, exploratory writing, legacy renders.

## What lives here

The leading underscore signals **historical / non-canonical** to outside readers. Personal correspondence (letters, exchanges with named collaborators), interview prep, and a legacy PDF dump live here. Exploratory pieces that may have informed canonical docs in [`../research/`](../research/) or [`../concepts/`](../concepts/) but never crystallized into one. Useful for reconstructing the paper trail; not part of the public protocol story. Treat correspondence with discretion — much of it is private by default.
