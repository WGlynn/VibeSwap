# Governance

> VIPs, VSPs, proposals, regulatory engagement.

## What lives here

Formal governance artifacts and the regulatory paper trail. VIPs (VibeSwap Improvement Proposals) and VSPs (VibeSwap Standards Proposals) are numbered, durable specifications; `proposals/` holds drafts and legacy proposals. `regulatory/` contains SEC engagement materials and compliance analyses. `ungovernance/` holds specs that intentionally constrain governance (timebomb, post-2026 ungovernance vision). The augmented-governance hierarchy — Physics > Constitution > Governance — is documented in [`../architecture/AUGMENTED_GOVERNANCE.md`](../architecture/AUGMENTED_GOVERNANCE.md).
