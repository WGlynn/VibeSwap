## 1. Install on iPhone

**Add VibeSwap to your home screen for an app-like experience:**

1. Open **https://frontend-jade-five-87.vercel.app** in Safari
2. Tap the **Share** button (square with arrow pointing up)
3. Scroll down and tap **"Add to Home Screen"**
4. Tap **"Add"** in the top right

That's it! VibeSwap now appears as an app on your home screen.

---

## 2. Install on Android

1. Open **https://frontend-jade-five-87.vercel.app** in Chrome
2. Tap the **three dots** menu (top right)
3. Tap **"Add to Home screen"**
4. Tap **"Add"**

---

## 3. Desktop Browser

Just bookmark **https://frontend-jade-five-87.vercel.app** or visit anytime. No installation needed.

---

## 4. For Developers

```bash
# Clone the repo
git clone https://github.com/WGlynn/VibeSwap.git
cd vibeswap/frontend

# Install dependencies
npm install

# Run locally
npm run dev
```

Opens at **http://localhost:5173**
