## The Three Meta-Primitives

Every paper in this collection derives from one of three meta-primitives:

1. **Separation** -- Separate things that don't belong together
2. **Composition** -- Compose things that amplify each other
3. **Alignment** -- Make the desired behavior the rational behavior

These are the intellectual foundation of Cooperative Capitalism and of any system that wants to be fair, robust, and composable.
