# Research

> Whitepapers, theorems, formal proofs, signature essays, and manifestos.

## What lives here

The signed, dated, citation-grade work. Whitepapers carry product/protocol claims; theorems carry named results; proofs are the formal mathematics; essays are the canonical voice pieces. If you want to *cite* VibeSwap, this is where to find the durable artifact. For descriptive design write-ups, see [`../architecture/`](../architecture/); for individual primitives in isolation, [`../concepts/`](../concepts/).

## Subfolders

- `whitepapers/` — protocol/product whitepapers (VIBESWAP, INTRINSIC_ALTRUISM, WALLET_RECOVERY, INCENTIVES, REPUTATION_ORACLE)
- `papers/` — 100+ papers across mechanism design, consensus, oracles, governance (see `papers/README.md` if present)
- `proofs/` — formal proofs and mathematics (Lawson Floor, IIA empirical verification, fairness)
- `theorems/` — named theorems (Fairness Fixed Point, Revenue Separation, Possibility, Provenance, Transparency, Trinomial Stability)
- `essays/` — signature essays and manifestos (Cave, Inversion, Hard Line, Recursion of Self-Design, etc.)
