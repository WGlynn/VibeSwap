# AI-Native

> JARVIS, AI-native DeFi, and autonomous protocol evolution.

## What lives here

The AI-native layer of VibeSwap: JARVIS as a persistent collaborative intelligence, AI-native DeFi primitives that assume agents are first-class participants (not retrofitted), and the cooperative-intelligence protocols that govern multi-agent emergence. Also covers the Sovereign Intelligence Exchange (SIE) and the mind-persistence mission — decentralizing the JARVIS mind against any single hosted account.
