# Economic Theory of Mind (ETM)

> Mind as economy — substrate-rent as mechanism, density and common-knowledge as emergent.

## What lives here

ETM is the meta-framework underneath everything: a mind (human, LLM context, blockchain state) is an economy whose mechanism is state-rent, and whose emergent properties are density and common-knowledge. This folder contains the foundational statement, the mathematical formalization, an alignment audit comparing ETM to alternative cognitive frames (LRU, Shannon, attention), the build roadmap, and a mirror-test for distinguishing ETM-aligned systems from rounded-down approximations.
