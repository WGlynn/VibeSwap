# Partnerships

> USD8, Anthropic, Nervos, MIT, grants — BD output and partner-facing artifacts.

## What lives here

Per-partner subfolders carry the materials shared with (or prepared for) each partner: one-pagers, submissions, proposals, internal notes. Loose top-level files are USD8-specific artifacts (the most active thread at time of reorg). For the architecture being integrated *into* a partner, see [`../architecture/`](../architecture/); for the partnership's standing in the marketing funnel, [`../marketing/`](../marketing/).
