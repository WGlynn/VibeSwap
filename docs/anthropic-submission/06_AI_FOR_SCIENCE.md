## Research Area
Mechanism Design, Game Theory, and AI Coordination in Decentralized Financial Systems

## Project Description
VibeSwap is both a deployed protocol and an active research project producing novel contributions at the intersection of game theory, cryptography, and AI coordination. We've published 23 research papers covering:

- **Shapley Value Distribution in DeFi** — Fair attribution using cooperative game theory for both human and AI contributors
- **Commit-Reveal Batch Auctions** — MEV elimination through cryptographic commitment schemes with uniform clearing prices
- **Trinomial Stability Theorem** — Novel stability mechanism combining PI controller, proof-of-work, and elastic rebase
- **Proof of Mind** — Verifiable AI individuality through cognitive evolution chains
- **Near-Zero Token Scaling** — Cost model where AI network overhead scales with users, not shards (4% overhead for full BFT+CRPC)
- **Contribution Yield Tokenization** — Pendle-inspired separation of idea value (permanent) from execution value (time-bound)
- **PsiNet/ERC-8004** — AI agents as first-class DeFi citizens with delegatable identities

## How API Credits Would Be Used
- JARVIS (Claude-powered) runs 24/7 on Fly.io serving real users
- Current inference costs ~$5/day across 3 BFT nodes
- Additional credits would enable:
  - Scaling to more users without compute budget constraints
  - Deeper research into AI coordination patterns under economic incentives
  - Extended Proof of Mind documentation (more sessions = more evidence)
  - Multi-agent experiments (multiple Claude-powered agents in adversarial/cooperative scenarios)

## Impact
23 papers already published. 71 knowledge primitives extracted. All open source. API credits directly translate to more research output — every inference JARVIS makes is a data point in the study of bounded AI autonomy under real economic constraints.
